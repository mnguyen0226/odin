### How to run Jest
In each directory install
```bash
npm init
npm install --save-dev jest
```

Make a copy to js
```json
{
  "name": "hello-world",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "scripts": {
    "test": "jest"
  },
  "keywords": [],
  "author": "",
  "license": "ISC",
  "devDependencies": {
    "jest": "^26.6.3"
  }
}
```

Run test
```
mnguyen0226@pop-os:~/Documents/personal/repositories/software_projects/odin/javascripts/arrays/repeat_string$ npm test
```
